# 🌾 Agentic AI System for Precision Agriculture

## 📖 Overview

The **Agentic AI System for Precision Agriculture** is an intelligent farming assistant that leverages Large Language Models (LLMs), AI agents, and machine learning to provide smart agricultural recommendations. The system helps users with crop analysis, fertilizer recommendations, disease detection, and agricultural guidance through an interactive chatbot and web interface.

---

## 🚀 Features

- 🌱 AI-powered agricultural assistant
- 🤖 Multi-agent workflow using LangChain
- 💬 Intelligent chatbot with OpenAI and Ollama
- 🌾 Crop analysis and recommendations
- 🧪 Fertilizer recommendation
- 📊 Interactive Streamlit dashboard
- ⚡ REST APIs using Flask and FastAPI
- 📚 Knowledge retrieval using FAISS vector database

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/delindavid/agentic-ai-system-for-precision-agriculture.git
```

Navigate to the project directory:

```bash
cd agentic-ai-system-for-precision-agriculture
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
